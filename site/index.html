Por que usar o Memo

🔒

Criptografia forte

AES-256-GCM com chave derivada via PBKDF2-SHA256 (200.000 iterações). Cada segredo é um arquivo cifrado.

💻

GUI + linha de comando

Interface gráfica para o dia a dia e comandos memo get / memo set para automação.

⚡

Busca instantânea

Encontre qualquer segredo digitando. Enter copia o valor direto para a área de transferência.

🔁

Atualização automática

O app verifica novas versões no GitHub e avisa você. Um clique para atualizar e reiniciar.

🏠

100% local

Sem servidores, sem telemetria, sem cadastro. Seus dados ficam só na sua máquina (e no seu backup).

⏱️

Sessão temporária

Depois de destrancado, o cofre fica acessível por uma janela curta — sem redigitar a senha toda hora.

Como funciona

1

Crie seu cofre

No primeiro uso, defina uma senha-mestra. Ela protege todos os documentos e não pode ser recuperada se esquecida.

2

Guarde seus segredos

Adicione chaves e valores pela interface ou pela linha de comando. Cada item vira um arquivo criptografado.

3

Acesse quando precisar

Busque, copie e edite com um clique. Pela CLI: memo get "github token" já copia o valor.

Segurança em primeiro lugar

O Memo foi desenhado para que seus dados sejam inúteis para qualquer um sem a senha-mestra. A senha nunca é gravada em disco — apenas a chave de sessão temporária, protegida pela DPAPI do Windows (vinculada à sua conta de usuário).

  • Cifragem autenticada AES-256-GCM (detecta adulteração).
  • Derivação de chave PBKDF2-SHA256 com 200.000 iterações.
  • Sessão isolada por cofre e validada contra o arquivo de configuração.
  • Executável assinado e verificado por SHA256 antes de cada atualização.
PowerShell
# copia um segredo para a área de transferência
memo get "github token"

# cria ou atualiza um segredo
memo set "chave pix" = a1b2c3d4

# abre a interface gráfica
memo
